## Live Site

See the live site at [https://paddletotheseacottages.com/](https://paddletotheseacottages.com/).

## Installing Deps

```
cd site
bundle install
```

## Build Site

```
cd site
bundle exec jekyll build
```

## Test Site

```
cd site
bundle exec jekyll serve
```
